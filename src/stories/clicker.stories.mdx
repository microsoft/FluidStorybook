# Clicker Demo

This example demonstrates how to create a clicker data object used in real-time collaboration. The data object
uses the `SharedCounter` DDS located in the `@fluidframework/counter` package to keep track of clicks
made by different users. `SharedCounter` allows for whole numbers to be incremented.

Key Distributed Data Structures (DDSs) used include:

* SharedCounter
